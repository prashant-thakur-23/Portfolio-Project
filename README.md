## ✨ Features

### 🎨 **User Interface**
- **Dark/Light Theme** - Toggle between dark and light modes
- **Responsive Design** - Optimized for desktop, tablet, and mobile
- **Smooth Animations** - Scroll reveals, typing effect, particle background
- **Interactive Elements** - Hover effects, ripple buttons, mouse glow

### 📊 **Dynamic Content**
- **Live GitHub Data** - Fetches real-time profile, repositories, and languages
- **Project Filtering** - Filter projects by category and search
- **Typing Animation** - Dynamic text rotation in hero section
- **Scroll Progress** - Visual indicator of scroll position

### 🎯 **Sections**
- **Hero** - Introduction with typing effect and profile photo
- **About** - Personal bio and traits
- **Skills** - Technical skills with interactive cards
- **Projects** - 11+ projects with filter and search
- **GitHub** - Live GitHub profile and repositories
- **Learning** - Currently learning technologies
- **Education** - Academic details
- **Certifications** - Professional certifications
- **Contact** - Contact form with validation

### ⚡ **Performance Optimizations**
- **Caching** - GitHub data cached for 5 minutes
- **Debouncing** - Search input optimized
- **Lazy Loading** - Images load on demand
- **CSS Animations** - GPU accelerated
- **DOM Fragments** - Batch DOM updates

## 🛠️ Technologies Used

### **Frontend**
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid, Animations
- **JavaScript (ES6+)** - Vanilla JS, Async/Await, DOM Manipulation

### **APIs & Services**
- **GitHub REST API** - Real-time data fetching
- **Google Fonts** - Inter & Space Grotesk fonts

### **Tools & Features**
- **LocalStorage** - Theme preference persistence
- **SessionStorage** - API data caching
- **Intersection Observer** - Scroll animations
- **Fetch API** - HTTP requests
- **Canvas API** - Particle background
